## Chrome Web Store Privacy & Permissions Answers

**Single Purpose Description**
Zen Task New Tab is a minimalist dashboard that replaces the new tab page to help users focus on a single task at a time, complete with a Pomodoro timer and mindfulness reminders.

**Permission Justification**

*   **`storage`**: Required to save the user's current active task, timer settings, and preferences locally. This ensures that user data persists between browser sessions and tab reloads.
*   **`alarms`**: Required to power the background focus timer. It ensures the countdown continues accurately and reliably even when the new tab page is closed or the browser is minimized.
*   **`notifications`**: Required to alert the user when a focus session or break timer is complete, ensuring they don't miss the end of a session while working in other tabs.

**Remote Code**
No. This extension is self-contained and does not load or execute remote code.
